Raytracer

Using equations and tips from an incredibly helpful blog by Mitchell Kember, I made a simple raytracer in Python. Right now, it only handles diffuse reflection. In the future, I hope to add specular reflection, anti-aliasing, and transmittance. I'm currently looking into using parallelization to speed up render times.

The repo includes progress pics and my attempt at converting the code to C++. Raytracer.py is the current working program.
